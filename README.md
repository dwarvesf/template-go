# template-go

> golang template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao go my-project
```

### From git

```bash
sao dwarvesf/template-go my-project
```
