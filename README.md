# template-go

> Template for scaffolding your next Go http server project, based on [gokit](https://github.com/go-kit/kit).

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
sao dwarvesf/template-go my-project
```
